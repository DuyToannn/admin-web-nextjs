<h2 id="technologies">💻 Technologies</h2>

- Frontend : Typescript, NextJS, Ant Design, NextAuth
- Backend : NestJS
- Database : MongoDB
- Deploy : Docker
- Dropbox API, Cloudinary API,JWT
